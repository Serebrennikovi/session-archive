# SA-T040_s02_sessions_index — Каталог сессий в браузере

## Customer-facing инкремент

Пользователь видит в браузере список сессий и может быстро отфильтровать историю по проекту, агенту, ветке, модели и дате.

---

## Scope

- Страница `/sessions` с таблицей или списком карточек
- Фильтры по `repo_name`, `branch`, `agent_family`, `ai_model`, диапазону дат
- Сортировка по `created_at` и `msg_count`
- Пагинация для больших архивов

## Out of scope

- Full-text search
- Saved views
- Inline review actions

---

## Как протестировать

1. Выполнить SQLite sync
2. Открыть `/sessions`
3. Применить несколько фильтров
4. Переключить сортировку и пройти по страницам пагинации

---

## Критерии приёмки

1. `/sessions` показывает данные из PostgreSQL read model
2. Все заявленные фильтры работают совместно
3. Сортировка не ломает пагинацию
4. Переход на страницу детали доступен из каждого элемента списка
